# 拾句 — 项目交接文档

## 一句话
输入一个词（如"孤独""夏天""故乡"），AI 从唐诗宋词古文散文中找到最匹配的古典名句，生成一张诗意卡片。

---

## 技术栈
- **前端**：Streamlit (Python)
- **大模型**：DeepSeek-V3 + Qwen2.5-VL-32B（通过硅基流动 API）
- **API 格式**：OpenAI SDK 兼容
- **部署**：Streamlit Cloud（免费）

## 四种风格（在 prompts.py 的 STYLES 字典里）

| 风格 | 引用来源 | 底色 |
|------|---------|------|
| 唐诗气象 | 李白/杜甫/王维/白居易 | 宣纸黄 |
| 宋词余韵 | 苏轼/李清照/辛弃疾/柳永 | 淡紫 |
| 古文雅意 | 归有光/韩愈/张岱/沈复 | 古纸棕 |
| 散文光阴 | 朱自清/汪曾祺/余光中/沈从文 | 暖黄 |

每张卡片结构：
```
[古典原文] ← AI 从古典文献中匹配
— 作者 · 篇名
[白话解读] ← AI 写的简短解读
[情绪印章]
```

## 核心逻辑

### 用户操作流程
1. 输入关键词 → 选风格 → 点「生成」
2. agent.generate_card() 调用 DeepSeek API
3. 返回 JSON {quote, source, paraphrase, emotion}
4. render_card() 渲染成 HTML 卡片

### API 调用链
app.py → agent.py → llm_client.py → SiliconFlow API (OpenAI SDK)
